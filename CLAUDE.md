# photoLearn - HarmonyOS Photo Learning App

## Project Overview
This is a HarmonyOS application built with ArkTS/ArkUI for learning photo-related functionality.

## Project Structure
- `entry/src/main/ets/` - Main application source code
- `entry/src/main/resources/` - Application resources
- `AppScope/` - Application-level configuration
- `oh-package.json5` - Project dependencies

## Development
- Framework: HarmonyOS ArkTS/ArkUI
- Testing: Hypium framework
- Mocking: Hamock framework

## Build Commands
Use HarmonyOS DevEco Studio or hvigor build system:
- Build: Run through DevEco Studio
- Test: Use Hypium test runner

## Entry Points
- Main entry: `entry/src/main/ets/pages/Index.ets`
- Entry ability: `entry/src/main/ets/entryability/EntryAbility.ets`
